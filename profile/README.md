# Ley Abierta

**Legislación abierta y accesible para todos.**

Ley Abierta es un motor open source que descarga legislación oficial, la convierte en archivos Markdown versionados con Git — donde cada reforma es un commit con su fecha histórica — y expone esos datos mediante una API y una web para que cualquier ciudadano pueda buscar, comparar versiones y entender cómo cambian las leyes que le afectan.

## Repositorios

| Repo | Descripción |
|------|-------------|
| [leyabierta](https://github.com/leyabierta/leyabierta) | Código fuente: pipeline + API + web (TypeScript, Bun) |
| [leyes](https://github.com/leyabierta/leyes) | Legislación española en Markdown — cada reforma es un commit |

## Principios

- **Open source para siempre.** Sin monetización. Sin paywalls.
- **Ciudadano primero.** Accesible para todos, no solo abogados o desarrolladores.
- **Transparencia.** Las leyes pertenecen a los ciudadanos y su evolución debe ser visible.

## Stack

TypeScript · Bun · Astro · Elysia · SQLite · Biome

## Datos

Más de 12.000 leyes consolidadas españolas (1835–presente). Fuente: [BOE (Agencia Estatal Boletín Oficial del Estado)](https://www.boe.es/).
